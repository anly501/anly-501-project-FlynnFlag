What I have done with my codes?

1.Raw data:"rumor.csv" --> Codes:"data_cleaning_python.ipynb" --> Modified data:"labeled_rumor_python.csv"

There are mutiple languages in the data set,so I firstly drop the unstructured data which is not written in English. Since the data set is collected by crawled during a certain period, so I drop the date of crawling and in case it will cause skew in the model trained.Columns like id, img_url,spam_score are useless, we need to drop them. The values of ord_in_thread are 0 for all rows, which is meaningless. We also drop it. I transformed the value of author as 0/1. 0 means anonymous, while 1 means the name of writer is recorded. All Na values were also be considered as anonymous. We cannot replace data with missing value of unstructed data. There is not a properly way to fill na in country.So we just drop certain rows. Finally we delete the rows where its type is "satire","junksci" and "state" to ensure it is a rumor. This will increase the quality of the data. Used countervectorized create the model and fit the model to the data. Changed the dtype to int32 to decrease the size of data to store it.

2.Raw data:"truth.csv" --> Codes:"data_cleaning_r" --> Modified data:"labeled_truth_R.csv"

Firstly,I transforms cities to country. There are urls at the end of every tweet, so I write a normal expression to drop them. Finally, I added a label.
