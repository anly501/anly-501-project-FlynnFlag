Introduction

In this page, Naive Bayes will be used to build Rumor Detection. However, different features will be selected to build this model by python and R respectively. For python, rumor text, friends count, followers count, retweet count, favorite count and account description are used. For R, only quantitative features including friends count, followers count, retweet count and favorite count are considered.

Theory

The Naive Bayes classification algorithm is a probabilistic classifier. It is based on probability models that incorporate strong independence assumptions. Text classification is one of the strongest characteristics of the Naive Bayes classifier.
