Rumor dataset

This data set is downloaded from public open source. It contains text and metadata scraped from 244 websites tagged as "bullshit" by the BS Detector Chrome Extension. This will be part of our labeled data to train a model.The dataset contains text and metadata from 244 websites and represents 12,999 posts in total. The data was pulled using the webhose.io API.

Rumors on Twitter

This is a collection of ruomrs on tweet. This data set is collected and labeled manually. All rumor tweets are found from the fact-check reports on official website of "Reuters", because they always put links of the source of the rumors in their reports. What I did is clicking the links and write down the text, the interaction number of that rumor.
